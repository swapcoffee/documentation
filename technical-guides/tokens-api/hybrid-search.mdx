Hybrid search queries jettons across two sources at once:

- **DEX-listed jettons** ‚Äî standard jettons tradeable through swap.coffee routing.
- **Memepad jettons** ‚Äî pre-DEX jettons still living on a bonding-curve launchpad (for example [TONFUN](https://tonfun.me/)). Once a memepad jetton's bonding curve graduates, the jetton transitions to a regular DEX-listed jetton.

This is the recommended endpoint to power a universal *"find a token"* input in your UI.

---

## Endpoint

```http
GET /api/v3/hybrid-search
```

**Query parameters:**

- **search** *(string)* ‚Äî free-text query across jetton name and symbol.
- **verification** *(array, default: `[WHITELISTED]`)* ‚Äî verification statuses to include. Accepted values: `WHITELISTED`, `COMMUNITY`, `UNKNOWN`, `BLACKLISTED`. Pass multiple values by repeating the parameter.
- **kind** *(enum)* ‚Äî scope of the search:
  - `ALL` ‚Äî everything
  - `DEXES` *(default)* ‚Äî only jettons listed on DEXes
  - `MEMES_ALL` ‚Äî all meme jettons (DEXes + memepads)
  - `MEMES_DEXES` ‚Äî meme jettons already graduated to DEXes
  - `MEMES_MEMEPADS` ‚Äî meme jettons still on memepads
- **sort** *(enum)* ‚Äî ordering: `FDMC`, `TVL`, `MCAP`, `VOLUME_24H`, `PRICE_CHANGE_24H`.
- **page** *(integer, 1‚Äì100, default: 1)*.
- **size** *(integer, 1‚Äì100, default: 20)*.

‚öÝÔ∏è **Note:** `page` and `sort` are ignored for some `kind` values where ordering is implicit. Refer to the [Swagger UI](https://tokens.swap.coffee/api/v3/swagger-ui) for the exact behavior per kind.

---

## Response

`200 OK` ‚Äî array of polymorphic jetton objects, discriminated by a `type` field:

- `type: "common"` ‚Üí a standard [jetton object](/technical-guides/tokens-api/jettons#jetton-object).
- `type: "memepad"` ‚Üí a [memepad jetton](#memepad-jettons).

## Memepad Jettons

A memepad jetton has a smaller shape than a DEX-listed jetton, because the usual market statistics (TVL across multiple pools, holders count, multi-window price changes) do not apply while the bonding curve is still live.

### Top-level fields

- **address** *(string, required)*: Raw jetton master address.
- **protocol** *(string, required)*: Memepad protocol identifier, e.g. `TONFUN`.
- **total_supply** *(string, required)*
- **decimals** *(integer, required)*
- **mintable** *(boolean, required)*
- **verification** *(enum, required)*: Same enum as for a regular jetton.
- **created_at** *(datetime, required)*
- **name** *(string)*
- **symbol** *(string)*
- **image_url** *(string)*
- **market_stats** *(object)* ‚Äî see below.

### `market_stats` for memepad jettons

- **price_usd** *(number)* ‚Äî current price in USD on the curve.
- **tvl_usd** *(number)* ‚Äî total value locked in the curve.
- **fdmc_usd** *(number)* ‚Äî fully diluted market cap in USD.
- **collected_ton** *(number)* ‚Äî TON already collected toward graduation.
- **max_ton** *(number)* ‚Äî TON target for graduation.
- **progress** *(number, 0‚Äì1)* ‚Äî share of the bonding curve completed.

Once a memepad jetton graduates, subsequent calls return it as a regular [jetton object](/technical-guides/tokens-api/jettons#jetton-object) with the full market statistics.
